# دليل مشروع محفظة الأعمال (Portfolio Project Guide)

هذا الملف يحتوي على شرح شامل لمشروع محفظة الأعمال (بورتفوليو) الخاص بك، وكيفية التعديل عليه، وطريقة إضافة أعمال ومشاريع جديدة في المستقبل بسهولة.

---

## 🏗️ هيكل المشروع وبنيته
تم بناء هذا المشروع باستخدام أحدث تقنيات الويب ليكون سريعاً، متجاوباً، وذو طابع سينمائي.
- **إطار العمل:** React (عن طريق Vite)
- **التصميم وتنسيق العناصر:** Tailwind CSS
- **الحركات والتأثيرات (Animations):** مكتبة Motion (Framer Motion)
- **التمرير الناعم (Smooth Scrolling):** مكتبة Lenis
- **لغة البرمجة:** TypeScript

تم تقسيم المشروع إلى عدة أقسام (Components) ليسهل التعديل عليها، وتجد كل قسم في المسار:
`src/components/`

أهم الأقسام:
- `Preloader.tsx`: شاشة التحميل الافتتاحية للموقع.
- `HeroSection.tsx`: الواجهة الرئيسية التي تحتوي على صورة الشخصية، والخلفية السينمائية، والنص الترحيبي.
- `WorkSection.tsx`: قسم استعراض الأعمال السابقة والمشاريع التي تظهر وتتحرك بسلاسة عند التمرير.
- `AboutSection.tsx`: قسم النبذة الشخصية (من أنا).
- `SkillsSection.tsx`: قسم المهارات والتخصصات.
- `ToolsSection.tsx`: قسم الأدوات المستخدمة (Photoshop, Illustrator, ...).
- `ContactSection.tsx`: قسم التواصل.
- `WhatsAppWidget.tsx`: الزر العائم للتواصل المباشر عبر واتساب.
- `GlobalMouseGlow.tsx`: تأثير مؤشر الماوس المخصص والإضاءة التفاعلية في الخلفية.

## 🖌️ كيفية التعديل على محتوى الموقع الحالي

### 1. تعديل الصور الرئيسية
- **صورة الخلفية الرئيسية:** تم تحديدها في ملف `src/App.tsx` ويمكنك تغيير الرابط الخاص بها في عنصر الـ `style`. وقد تم رفعها على Cloudinary للحصول على أفضل أداء.
- **صورة الشخصية:** موجودة داخل `src/components/HeroSection.tsx` وتعتمد أيضاً على رابط من Cloudinary.

### 2. تعديل النصوص الثابتة 
العديد من النصوص موجودة مباشرة داخل الـ Components الخاصة بها.
- لتغيير أو إضافة مهارات وأدوات، قم بفتح `SkillsSection.tsx` أو `ToolsSection.tsx` وستجد مصفوفة (Array) أعلى الملف تحتوي على الأسماء، يمكنك تعديلها أو إضافة المزيد ببساطة.
- لتغيير معلومات التواصل، افتح `ContactSection.tsx` وقم بتعديل النصوص والروابط.

### 3. تعديل رقم الواتساب الخاص بالزر العائم
- افتح ملف `src/components/WhatsAppWidget.tsx`.
- ستجد الرابط `href="https://wa.me/201234567890"`.
- قم بتغيير الرقم `201234567890` ليطابق رقم الواتساب الخاص بك بالمفتاح الدولي (بدون علامة + أو أصفار إضافية).

## 🎨 كيفية إضافة أعمال (مشاريع) جديدة لمعرض الأعمال

يوجد ملف منفصل مخصص لإدارة جميع مشاريعك وأعمالك وهو متواجد في المسار التالي:
`src/data/projects.ts`

بمجرد إضافة مشروع جديد في هذا الملف، سينعكس تلقائياً على الموقع ليظهر ضمن عرض بطاقات المشاريع بطريقة رائعة وبنفس الحركات.

### طريقة إضافة مشروع:
افتح ملف `src/data/projects.ts`. ستجد مصفوفة تسمى `projects` تحتوي على كائنات (Objects). لإنشاء مشروع جديد، كل ما عليك هو إضافة كائن جديد في بداية أو نهاية المصفوفة، بهذا الشكل:

```typescript
{
    id: 9, // يجب أن يكون رقماً فريداً غير مكرر مع باقي المشاريع
    title: "اسم المشروع الجديد", // عنوان المشروع مثلا PORTFOLIO DESIGN
    category: "تصميم سوشيال ميديا", // التخصص
    client: "اسم العميل أو الشركة", 
    role: "دورك في المشروع، مثلا: مصمم رئيسي",
    year: "2026",
    image: "رابط الصورة المصغرة التي ستظهر في البطاقة",
    color: "#ff5500", // اللون الذي سيشع كخلفية عند فتح المشروع
    description: "وصف قصير عن المشروع والهدف منه",
    content: [
      "الفقرة الأولى التفصيلية عن المشروع.",
      "الفقرة الثانية إن وجدت.."
    ],
    gallery: [
      "رابط صورة إضافية 1",
      "رابط صورة إضافية 2"
    ] // يمكنك إضافة العديد من الصور لاستعراض كامل المشروع عند النقر عليه
}
```

**ملاحظات عند إضافة مشاريع:**
- تأكد من عدم تكرار الـ `id`.
- ارفع الصور الخاصة بك على موزع صور مثل (Cloudinary أو صور جوجل أو مجرد مسار داخلي) وضع الروابط في متغير `image` للغلاف، ومتغير `gallery` لصور معرض المشروع.
- قم بوضع كود اللون في الخاصية `color`، هذا اللون سيتم استخدامه لعمل تأثير الإضاءة والتوهج المخصص للمشروع الخاص بك عندما يقوم الزائر بتكبيره وقراءته.

## ⚙️ ملاحظات برمجية وأداء (للمتخصصين)
- **أداء التمرير:** مساحة الـ Scroll الأفقية في قسم معرض الأعمال تُنظم نفسها طردياً بناءً على عدد المشاريع داخل ملف `projects.ts`، ويتم استخدام خصائص التسريع العتادي GPU (`transform-gpu` و `will-change`) لضمان سلاسة التمرير وعرض التأثيرات بشكل ممتاز دون تقطيع.
- **الحركات:** الحركات والانميشن الموجودة في الموقع تتم عبر مكتبة Motion، ولتعديل السرعة أو الزوايا يمكن تغيير خصائص `transition`. 
- **مؤشر الماوس المخصص:** تم إخفاء الـ Cursor الافتراضي وتفعيل مؤشر مخصص يختفي على الأجهزة التي تعمل باللمس لضمان تجربة مستخدم مثالية على هواتف المستخدمين، يتواجد هذا التحكم داخل قسم `GlobalMouseGlow.tsx`.

مع خالص الأمنيات بإطلاق معرض أعمال مميز وناجح! 🚀
